# Service Images Placeholders

Add the following images to this folder for the service booking form:

## Required Images:
1. `bike-placeholder.png` - Image for Bike and Rickshaw Rental
2. `guided-placeholder.png` - Image for Guided Tours of Lucca
3. `hills-placeholder.png` - Image for Trips in the Tuscan Hills
4. `coach-placeholder.png` - Image for Coach Trip Packages
5. `luxury-placeholder.png` - Image for Transportation with Luxury Cars
6. `wine-placeholder.png` - Image for Wine Tours by Bus
7. `default-placeholder.png` - Default image shown before any service is selected

## Recommended Image Specifications:
- Format: PNG or JPG
- Size: 800x600 pixels minimum
- Aspect ratio: 4:3 or similar
- File size: Under 500KB for fast loading

## Tips:
- Use high-quality images that represent each service
- Consider using images with similar lighting/style for consistency
- The images will be displayed in a container with `object-fit: cover`
